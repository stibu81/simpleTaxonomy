# Save Taxonomic Hierarchy to HTML-File

Save a plot of a taxonomic hierarchy created with
[`plot_taxonomy()`](https://stibu81.github.io/simpleTaxonomy/reference/plot_taxonomy.md)
to a self-contained HTML-file.

## Usage

``` r
save_taxonomy(widget, file, background = "white", title = "simpleTaxonomy")
```

## Arguments

- widget:

  a widget created with
  [`plot_taxonomy()`](https://stibu81.github.io/simpleTaxonomy/reference/plot_taxonomy.md).

- file:

  the full path to the file

- background:

  character giving the html background colour of the widget. Defaults to
  white.

- title:

  text to use as the title of the saved page
