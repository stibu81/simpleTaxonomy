# simpleTaxonomy: Create Interactive Visualisation of Taxonomic Hierarchy

Create interactive visualisations of a taxonomy hierarchy that is
specified in a csv file using a simple format. The visualisation uses
the javascript library D3.js via the R package collapsibleTree.
